# Dotfiles

Ice + Rose theme dotfiles.

## Install

```bash
./install.sh && source ~/.zshrc
```

## Uninstall

```bash
./uninstall.sh
```

## Structure

- `kitty/` - Kitty terminal (JetBrains Mono Nerd Font)
- `vim/` - Vim config + ice_rose colorscheme
- `zsh/` - Zsh prompt, aliases, syntax highlighting
- `theme/` - Color palette definitions
